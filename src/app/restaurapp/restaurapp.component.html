Restaurapp

Una aplicación FrontEnd desarrollada con HTML, CSS y JavaScript que permite tomar comandas haciendo uso de LocalStorage.
Se pueden modificar los datos de inicio de sesión de hasta 4 camareros y del menú (añadir nuevos artículos, modificar el nombre, precio y categoría)

Instrucciones de uso:

  1. Inicia sesión: Usuario "camarero1" y Contraseña "1234".
  2. Pulsa en el número para cambiar el estado de la mesa a "abierto".
  3. Pulsa en el número de la mesa para acceder al menú.
  4. Despliega las categorias y añade artículos a la comanda con las teclas + y -.
  5. Pulsa el boton añadir comanda para guardar los cambios.
  6. Puedes volver al dashboard de las mesas pulsando "Volver".
  7. Puedes generar el tícket pulsa "Generar cuenta".
  8. Puedes visualizar el tícket en el dashboard y acceder al pago. (TPV ficticio con validaciones)
  9. Accede al área de administrador con Usuario "admin" y contraseña "nimda".
  10. Modifica los datos de inicio de sesión o del menú.
